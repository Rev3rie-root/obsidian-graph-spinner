Full:
***
**Graph Spinner** is a lightweight Obsidian plugin that slowly rotates your graph view nodes in a smooth, galaxy-like motion — designed as a gentle visual aid for writers facing creative block. Instead of staring at a static web of notes, you can watch your vault drift like a star system, giving your mind something calming to rest on while ideas resurface naturally. The rotation speed is fully adjustable from the plugin settings, and the animation responds cleanly to graph filter changes like toggling orphan nodes. Built with requestAnimationFrame for a smooth, screen-synced experience with minimal performance impact.
